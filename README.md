# Sub-GS-RPA
Esse repositório vai conter o trabalho que foi feito por mim na Substitutiva da matéria de RPA

Devido a falta de arquivos para rodar os scripts e obter evidencias, pedi para o Claude gerar 3 arquivos ficticios contendo informações sobre temperatura, consumo de energia, posicionamento orbital e estado dos equipamentos embarcados. Os arquivos gerados se encaixam dentro do contexto do trabalho, fornecendo os dados nescessarios para que os scripts rodem sem problema algum. Espero que não tenha nenhum problema. Tudo isso foi feito dentro de um Ambiente Virtual (venv).
